## Adding New Screenshots

To add screenshots:

1. **Take Screenshot**: Use browser DevTools device mode for consistent dimensions
2. **Recommended Dimensions**: 
   - Full page: 1920x1080 (desktop)
   - Mobile: 375x812 (iPhone X)
   - Tablet: 768x1024 (iPad)
3. **Optimize Images**: Use PNG for UI screenshots, compress for web
4. **Naming Convention**: Use kebab-case (e.g., `login-page.png`, `admin-dashboard.png`)
5. **Add to README**: Reference in main README.md Screenshots section
